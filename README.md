# Animated HTML Portfolio Application built with custom in-line CSS design.
## A full dark-tech design with:
### 1. Floating particle network animation on the background
### 2. Custom animated cursor with a glowing ring that reacts to hover
### 3. Glitch text effect on your name in the hero
### 4. Scroll-reveal animations — every section fades up as you scroll
### 5. SVG brand logos for Python, TypeScript, Java, LinkedIn, GitHub, etc.
### 6. Tabbed skills section — click to switch between Languages, Frontend, Backend, Data, Cloud
### 7. Project cards with hover lift + gradient top border reveal
### 8. Timeline for work experience
### 9. Social media grid with icons for GitHub, LinkedIn, Twitter, Medium, Website & Email
### 10. A pulsing avatar placeholder ready for your photo
